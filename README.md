# SD_2020
## 2020 KNU Software Design(005) Team 6

- 프로젝트 명: Health Care Software for Public
- 팀원: 석혜린, 이창진, 허승호
* 역할
  * 석혜린: PM, GUI, 광고, 메인 화면 출력
  * 이창진: DB 생성 및 관리, 로그인, 결제
  * 허승호: 피진단자 기록 추가/삭제, 데이터 전송, 진단서 작성, 데이터 이미지화, 지각 알림, 긴급 알림
